## How to Run

### 1. Admin Dashboard
```bash
cd admin-dashboard
npm install
npm run dev
```

### 2. Backend API
```bash
cd backend

# Create virtual environment (only once)
python3 -m venv venv

# Activate virtual environment
source venv/bin/activate   # macOS / Linux
# venv\Scripts\activate    # Windows

# Install dependencies
pip install -r requirements.txt

# Start the server
python main.py
```

### 3. Customer Site
```bash
cd src
npm install
npm run dev
```
